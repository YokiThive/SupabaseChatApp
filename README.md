# Chat App Using Flask & Supabase

## Overview
This is a real-time chat application. It allow users to authenticate, send messages and images just like Discord. 
The goal of this project is to explore and learn backend development, Supabase for database integration, and deployment using new tools.

## App Features
- User authentication via email and Google SSO
- Real-time messaging
- Support various image type
- Data storage and retrieval using Supabase

## Tech Stack
- Frontend: HTML, CSS, Javascript
- Backend: Python, Flask
- Database: Supabase
- Deployment: Vercel

## Assisting Tools
Claude AI and LibreChat was used to assist in frontend development and improving naming convention

## What I learnt
- Flask integration with Supabase services
- Handling API responses
- Managing user authentication
- Managing session flow
- Using AI tools to speed up frontend development
- Differences between serverless and traditional hosting
